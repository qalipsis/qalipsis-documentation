= QALIPSIS Available Plugins

In this section, documentation is provided for currently available plugins that can be used with your QALIPSIS project.

This section will be updated as more plugins are available.

Note: Before using a plugin, confirm that the related dependency has been added to your project:

* Gradle - Groovy DSL: 
+
`implementation group:'io.qalipsis', name: 'plugin-XXX', version: '{revnumber}`

* Gradle - Kotlin DSL: 
+ 
`implementation("io.qalipsis:plugin-XXX:{revnumber}")`

[cols="1,3"]
|===

|  *Name* 
|  *Description*

|link:08b_cassandra.adoc[cassandra]
|Connects QALIPSIS to Apache Cassandra with a Datastax driver, using  `CqlSessionBuilder` to build sessions, and Cassandra Query Language (CQL) for queries.

|link:08c_elasticsearch.adoc[elasticsearch]
|Connects QALIPSIS to Elasticsearch using the REST APIs.

|link:08d_graphite.adoc[graphite]
|Waiting on info...

|link:08e_influxdb.adoc[influxdb]
|Connects QALIPSIS to InfluxDB using InfluxDbClientKotlin with Flux v.2.0.

|link:08f_jackson.adoc[jackson]
|Supports reading a file (JSON, XML or CSV) and deserializing the data to objects.

|link:08g_jms.adoc[jms]
|Connects QALIPSIS to a messaging platform that supports https://www.jcp.org/en/jsr/detail?id=914[JMS] (for example, https://activemq.apache.org/[ActiveMQ]).

|link:08h_kafka.adoc[kafka]
|Connects QALIPSIS to an Apache Kafka cluster.

|link:08i_mongodb.adoc[mongodb]
|Connects QALIPSIS to MongoDb using the driver from org.mongodb.

|link:08j_netty.adoc[netty]
|Supports TCP, UDP, HTTP and MQTT using the non blocking Netty library.

|link:08kl_r2dbc-jasync.adoc[r2dbc-jasync]
|Connects QALIPSIS to MariaDB, MySQL and PostgreSQL using the R2DBC drivers from jasync-sql.

|link:08m_rabbitmq.adoc[rabbitmq]
|Connects QALIPSIS to RabbitMQ.

|link:08n_redis-lettuce.adoc[redis-lettuce]
|Connects QALIPSIS to Redis using the Lettuce client.
|===
