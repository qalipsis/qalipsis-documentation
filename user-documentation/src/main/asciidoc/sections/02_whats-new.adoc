== In this Release
QALIPSIS 1.0 is the initial product release.



QALIPSIS 1.0 is designed with the scaleablility and simplicity required to perform load-tests and performance verification on both distributed  and stand-alone systems

With QALIPSIS 1.0:

*You are able to create scenarios with different branches and running with many minions.

*You can use its many core steps and plugins to fire load on your system and feed QALIPSIS with data to verify load strength.

QALIPSIS 1.0 runs standalone. A history of events and metrics can be logged or saved in Elasticsearch.

Future releases of QALIPSIS will include more plugins, more steps for statistics, and QALIPSIS as a cluster.
